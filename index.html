Custom-social-share-buttons

A JavaScript layer that allows you to easily add share links for popular social sites without having to load their associated JavaScript library.

View the Project on GitHub amk221/Custom-Social-Share-Buttons

Instructions

  1. Create some bog-standard hyperlinks
  2. Add a class name to their containing element
  3. Add the service data attribute to the element you wish to trigger the sharing. (e.g. facebook)
  4. Add the open graph meta tags
  5. Initialise the sharer new Sharer(".share-links");
<div class="share-links">
    <a href="https://twitter.com/share" data-service="facebook">
      Share on Twitter
    </a>
    ...etc...
</div>
